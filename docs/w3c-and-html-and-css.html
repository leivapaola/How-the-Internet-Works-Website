W3C, HTML & CSS

Definition

HTML (the Hypertext Markup Language) and CSS (Cascading Style Sheets) are two of the core technologies for building Web pages. W3C(World Wide Web Consortium)an international community that includes a full-time staff, industry experts, and several member organizations.

Example

Examples for HTML are all over this page and every website on the internet. The Different size fonts to the Black background was made through an HTML code as show above in the top of the page.

Why

Cascading Style Sheet(CSS) is used to set the style in web pages that contain HTML elements. It sets the background color, font-size, font-family, color, most of the property of elements on a web page.

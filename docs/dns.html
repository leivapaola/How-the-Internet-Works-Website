DNS

Definition

Domain Name System - The Internet's system for converting alphabetic names into numeric IP addresses.

Example

If a computer needs to communicate with the web server: example.net, your computer needs the IP address of the web server example.net. They are technical documents about computer networks. Like www.amazon.com

Why

The Domain Name System (DNS) is a central part of the internet, providing a way to match names (a website you're seeking) to numbers (the address for the website) without DNS everyone would have to remember random strings of number to access different websites, or at least Google's IP address.
